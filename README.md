# Placement Management System Frontend

A React-based frontend application for managing placement activities, built with Create React App and styled with Tailwind CSS.

## Features

- User authentication (Login page)
- Dashboard for overview
- Home page
- Additional pages: Not Found
- Components: Navbar, Footer, Loader, StatCard

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/Mohamed-kaleel/talentrix-frontend.git
   cd talentrix-frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm start
   ```

   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```
npm run build
```

## Available Scripts

- `npm start`: Runs the app in development mode
- `npm test`: Launches the test runner
- `npm run build`: Builds the app for production
- `npm run eject`: Ejects from Create React App (irreversible)
